For ICL -  The instrument needs a command to get the temperature data. 

The command        #WT.   <CR> <LF>


The command will return the 1st temperature is probe tip temperature and is a thermocouple.  The 2nd temperature is the probe housing temperature and is a thermistor.

The following is an example in docklight.
